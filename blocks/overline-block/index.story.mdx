# Overline

The `Overline` block is used by several blocks to display their top level section or relevant data.
Some of these blocks are:

- Top table list block
- Extra Large Promo block
- Large promo block

If no custom text and URL are passed as props to the Overline block, it will default to use data from global content.

## Use

To import it into a block:

```
import Overline from '@wpmedia/overline-block';

...
    <Overline
        customText="Overline text"
        customUrl="Overline url"
    />
...

```
